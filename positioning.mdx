---
title: Category Positioning
description: What WhyOps is, and what it is not.
---

WhyOps is not:
- Logging
- LLM tracing
- APM
- A monitoring dashboard

WhyOps is:
- Agent Runtime Observability
- Decision infrastructure for agents
- A reproducibility layer for agent state

Positioning statement:
- Observability shows what happened. WhyOps shows why.

Category comparison:
- Server logs to APM to distributed tracing to agent cognition observability.
- WhyOps represents the next abstraction shift.
