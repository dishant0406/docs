# Product Status

WhyOps is not yet shipped. There is no SDK or proxy available today.

<Callout type="warning">
These docs describe the architecture, event schema, and phased implementation plan. They are not integration instructions.
</Callout>

## What exists today

- A complete conceptual architecture
- Event schemas for agent cognition
- A phased execution roadmap

## What does not exist yet

- Proxy runtime
- SDK instrumentation
- Hosted UI
- APIs
